# Clean Temp Windows

## Scripts

### `clean_temp.bat`

Dọn dẹp file tạm (Temp) trên Windows — chạy với quyền Administrator.

**Tác dụng:**
- Xóa toàn bộ file trong `%TEMP%`
- Xóa toàn bộ file trong `C:\Windows\Temp`
- Dọn sạch Recycle Bin

**Cách dùng:**
```
Double-click file → tự động xin quyền Admin → chạy dọn dẹp.
```

Script sẽ tự động yêu cầu quyền Administrator nếu chưa được chạy ở chế độ Admin.
