# Developer Tools — Danh sách công cụ hữu ích

## Code Editor & IDE

### 1. Visual Studio Code
- **Trang chủ:** [code.visualstudio.com](https://code.visualstudio.com/)
- Editor đa năng cho mọi ngôn ngữ (JS, TS, Python, C#, Go, v.v.)
- Kho extension phong phú — Git, Docker, ESLint, Prettier, IntelliCode
- Integrated terminal, debugger, IntelliSense

### 2. Visual Studio
- **Trang chủ:** [visualstudio.microsoft.com](https://visualstudio.microsoft.com/)
- IDE chính cho .NET / C# — debug, profile, refactor mạnh nhất
- Git UI sẵn, xử lý conflict 3-way, tích hợp Azure DevOps

### 4. Notepad++
- **Trang chủ:** [notepad-plus-plus.org](https://notepad-plus-plus.org/)
- Text editor siêu nhẹ cho Windows — mở file lớn, syntax highlight
- Tìm kiếm regex, macro, plugin (JSON viewer, Compare, v.v.)

## API Testing

### 1. Postman
- **Trang chủ:** [postman.com](https://www.postman.com/)
- Gửi request HTTP (GET, POST, PUT, DELETE), kiểm tra response
- Collections, environments, pre-request/test scripts
- Tự động generate code từ request (curl, fetch, axios, v.v.)
- **Bản free đủ dùng** — Platform: Windows, macOS, Linux

### 2. Insomnia
- **Trang chủ:** [insomnia.rest](https://insomnia.rest/)
- API client nhẹ hơn Postman, giao diện sạch hơn
- Hỗ trợ GraphQL, gRPC
- **Miễn phí** — Platform: Windows, macOS, Linux

---

## Network & Debugging

### 1. Fiddler Everywhere
- **Trang chủ:** [fiddler.telerik.com](https://fiddler.telerik.com/)
- HTTP/HTTPS proxy debugging — xem, chặn, sửa request/response
- Bắt traffic từ trình duyệt, app desktop, mobile
- AutoResponder, throttling, request composer
- **Bản free có giới hạn** — Platform: Windows, macOS, Linux

## Database

### 1. DBeaver
- **Trang chủ:** [dbeaver.io](https://dbeaver.io/)
- Universal database tool — hỗ trợ MySQL, PostgreSQL, SQL Server, SQLite, Oracle, v.v.
- ER diagram, data export/import, SQL editor với autocomplete
- **Bản Community miễn phí** — Platform: Windows, macOS, Linux

### 2. SSMS (SQL Server Management Studio)
- **Trang chủ:** [docs.microsoft.com/sql/ssms](https://docs.microsoft.com/en-us/sql/ssms/)
- Tool chính thức cho SQL Server — query, admin, backup, restore
- Execution plan, index tuning, profiler
- **Miễn phí** — Platform: Windows

## Version Control & Diff

### 1. Git
- **Trang chủ:** [git-scm.com](https://git-scm.com/)
- Hệ thống quản lý phiên bản phân tán — bắt buộc với mọi developer
- **Miễn phí** — Platform: Windows, macOS, Linux

## Terminal & Shell

### 1. Windows Terminal
- **Trang chủ:** [github.com/microsoft/terminal](https://github.com/microsoft/terminal)
- Terminal hiện đại cho Windows — tab, pane, theme tuỳ chỉnh
- Hỗ trợ đồng thời PowerShell, CMD, WSL, Git Bash
- **Miễn phí** — Platform: Windows

### 2. PowerShell Core (7+)
- **Trang chủ:** [github.com/PowerShell/PowerShell](https://github.com/PowerShell/PowerShell)
- Cross-platform shell & scripting — mạnh hơn CMD
- **Miễn phí** — Platform: Windows, macOS, Linux

---

## Container & Virtualization

### 1. Docker Desktop
- **Trang chủ:** [docker.com/products/docker-desktop](https://www.docker.com/products/docker-desktop/)
- Chạy container — dev environment đồng nhất, CI/CD
- Docker Compose cho multi-container
- **Bản Personal free** — Platform: Windows, macOS, Linux

### 2. WSL 2
- **Trang chủ:** [learn.microsoft.com/wsl](https://learn.microsoft.com/en-us/windows/wsl/)
- Chạy Linux ngay trên Windows — không cần VM
- Tích hợp với VS Code (Remote - WSL)
- **Miễn phí** — Platform: Windows

---

## Message Queue & Cache

### 1. Redis
- **Trang chủ:** [redis.io](https://redis.io/)
- In-memory data store — dùng làm cache, session, message broker
- Hỗ trợ string, hash, list, set, sorted set, pub/sub
- **Miễn phí (open source)** — Platform: Windows (bản port), macOS, Linux
- **GUI tool:** RedisInsight ([redis.com/redis-enterprise/redis-insight](https://redis.com/redis-enterprise/redis-insight/))

### 2. KafkIO
- **Trang chủ:** [kafkio.com](https://kafkio.com/)
- Kafka GUI desktop — quản lý topic, message, consumer, ACL
- Tích hợp Schema Registry, ksqlDB, Kafka Connect
- Kết nối Confluent Cloud, Aiven, Azure Event Hubs, Amazon MSK
- **Miễn phí (donationware)** — Platform: Windows, macOS, Linux

## Developer Utilities

### 1. Everything
- **Trang chủ:** [voidtools.com](https://www.voidtools.com/)
- Tìm kiếm file tức thời trên Windows (nhanh hơn Windows Search gấp trăm lần)
- **Miễn phí** — Platform: Windows

### 2. Microsoft PowerToys
- **Trang chủ:** [learn.microsoft.com/powertoys](https://learn.microsoft.com/en-us/windows/powertoys/)
- Bộ tiện ích cho Windows: FancyZones (cửa sổ), PowerRename, Color Picker, Image Resizer, v.v.
- **Miễn phí** — Platform: Windows

### 3. ShareX
- **Trang chủ:** [getsharex.com](https://getsharex.com/)
- Chụp màn hình, quay màn hình, upload tự động
- **Miễn phí (open source)** — Platform: Windows

### 4. AutoHotkey
- **Trang chủ:** [autohotkey.com](https://www.autohotkey.com/)
- Auto script — tạo shortcut bàn phím, tự động hoá thao tác lặp
- **Miễn phí** — Platform: Windows
