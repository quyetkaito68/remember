# Game Engine

> Game Engine là gì, gồm những gì, và các chuẩn chung

---

## Engine thường gồm những gì?

```text
Engine

├── Renderer
├── Physics
├── Audio
├── Animation
├── Scene
├── Camera
├── UI
├── Input
├── Asset
├── Networking
├── AI
├── Lighting
├── Scripting
└── Editor
```

---

## Chuẩn chung của mọi Engine

Dù là Unity, Unreal hay Godot đều có các khái niệm gần giống nhau:

```text
Game

├── Scene
│
├── Camera
│
├── Object
│
├── Component
│
├── Script
│
├── Animation
│
├── Physics
│
├── Lighting
│
├── UI
│
└── Asset
```

Tên gọi có thể khác nhau nhưng tư duy gần như giống nhau.
